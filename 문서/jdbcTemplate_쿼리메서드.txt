<JdbcTemplate의 query 메서드의 종류>
1. query(String query, RowMapper<T> rowMapper)
: 다수의 행을 조회할 때 사용
: 결과를 List형태로 반환

2. query(String query, RowMapper<T> rowMapper, 매개변수1, 매개변수2, ....)
: 다수의 행을 조회하며 쿼리 파라미터(= ?)를 바인딩할 때 사용

3. queryForObject(String query, RowMapper<T> rowMapper, 매개변수1, 2, ....)
: 단일 행을 반환

4. queryForObject(String query, 데이터타입, 매개변수1, 2, ....)
: 단일 값을 반환 (int 라던지 String 등 단일 데이터타입)
: 스칼라 서브쿼리(단일 값을 리턴하는 서브쿼리)를 사용할때 주로  사용함
